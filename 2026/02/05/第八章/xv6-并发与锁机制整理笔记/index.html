3️⃣ 所以“函数规范”必须包含什么？

除了：

  • 参数
  • 返回值
  • 副作用

还必须包含：

  • 调用前允许持有哪些锁
  • 函数内部会获取哪些锁
  • 锁的相对顺序要求

📌 结论（面试金句）

在内核中，锁顺序是函数接口语义的一部分，而不仅是实现细节。

四、自旋锁 + 中断：为什么必须关中断

1️⃣ 典型死锁场景（单 CPU）

  • 中断等锁
  • 锁等中断返回
  • ⇒ 永远卡死

2️⃣ xv6 的解决方案（保守但安全）

CPU 一旦持有任何自旋锁，就关闭该 CPU 的中断

  • 中断仍可发生在其他 CPU
  • 不会产生自我等待

3️⃣ push_off / pop_off 的意义

  • 支持 嵌套临界区
  • 记录“最外层”进入临界区时的中断状态
  • 只有最后一个锁释放时才恢复中断
